Состав комплекса
================

Комплекс состоит из виртуальных машин выполняющих функции:

1. базы данных;
2. сервера приложений;
3. sip-сервера;
4. веб-сервера.

Каждый из типов представлен двумя виртуальными машинами организованных в кластер высокой доступности.
